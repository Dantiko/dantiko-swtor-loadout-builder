# Dantiko's SWTOR Loadout Builder

This app allows users to create, save, load, import, and export loadouts for their characters in SWTOR for later reference in-game.

Includes these features:

- **Class/Spec Selections** (they will auto update the stat calcs depending on selections)

- **At-A-Glance Gear Layout** (know exactly what is in each slot from easy to read color coding and text implementations)

- **Loadout Saving/Loading within the app itself** (loadouts save per character and include class/spec selections for reference at your convenience)

- **Loadout Import/Export** (to share saved loadouts with your friends)

- **Loadout and Character Management** (allows editing or deleting characters/loadouts)

- **Totals and Details Notes** (tells you what is included/excluded from calculations)
